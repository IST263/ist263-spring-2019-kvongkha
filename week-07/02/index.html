This is a good practice that will help you implement your wireframes.

Implement the above homepage design as a HTML page

Deliverables

  • Page Name - index.html (should go in the after-class folder.
  • Remember to use semantically correct HTML
  • Conside using Fontawesome Icons for 'Home' and 'Social Media'
  • Use the 'base.css' file from last class
  • If you need to write custom CSS put that in a separate css file

Some pointers to get you started

  • You don't have to create the 'x' that is in the boxes. That is just a wire-frame practice.
  • 90% of the CSS you will need is already available for you in 'base.css'

  • 1 - Good CSS organization
